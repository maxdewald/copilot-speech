# Security Policy

Please report vulnerabilities privately through GitHub Security Advisories for `maxdewald/copilot-speech`.

Do not include recorded audio, transcript text, credentials, or other sensitive material in an issue. Include extension version, operating system, VS Code version, helper version, and sanitized diagnostic output where relevant.

Production artifact installation will require pinned manifests, SHA-256 verification, safe extraction, atomic installation, and signed platform binaries. The draft manifests are not production trust roots.
